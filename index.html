Samuel L. Jackson Voice Transfer

The Project

Built for the summer 2019 summer hackathon, this project utilizes deep learning and transfer learning to convert a given speaker's vocal style into a target speaker's. It was pieced together using:
  1. Python 2.7 (Tensorflow, CUDA) via a Docker container
  2. deep-voice-conversion github repository
  3. A Rage in Harlem, by Chester Himes, narrated by Samuel L. Jackson
